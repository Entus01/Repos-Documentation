# 📘 Folleto Didáctico — Organización de Proyectos

## 📖 Descripción
Este repositorio funciona como un **folleto didáctico** en el que presento cómo organizo mis proyectos de desarrollo web.  
El objetivo es mostrar buenas prácticas de estructura, documentación y constancia en el flujo de trabajo, de manera que cualquier persona pueda entender y replicar el estilo de organización.

---

## 🛠️ Herramientas destacadas
- **React** — Librería para construir interfaces de usuario.  
- **Vite** — Bundler rápido y moderno para proyectos frontend.  
- **React Router** — Manejo de rutas y navegación entre páginas.  
- **JavaScript (ES6+)** — Lenguaje base del proyecto, sin TypeScript.  

---

## 🚀 Instalación y ejecución
```bash
# Clonar el repositorio
git clone https://github.com/Entus01/Repos-Structure.git

# Entrar al proyecto
cd Repos-Structure

# Instalar dependencias
npm install

# Ejecutar en modo desarrollo
npm run dev
```
